== Description ==

It will turn WordPress , bbPress and BuddyPress into Persian ! and also enables awesome features !

* **For Upgrading to 3.1 : Delete the plugin and install it again.**

* **برای به‌روزرسانی به نگارش ۳.۱ به بالا : نخست افزونه را پاک کنید و دوباره آن‌را نصب کنید .**


[توضیحات افزونه به زبان پارسی](http://chamrosh.ir/projects/persian-world "Chamrosh Studio")

= Compatibility =

*   WordPress : **3.9.x**
*   bbPress : **2.5.x**
*   BuddyPress : **2.0.x**


= Features =

*   It Translates **All Part of WordPress** , **bbPress** and **BuddyPress** and *Twentyfourteen* ( WordPress 3.8 default theme )
*   Enables Word Captions (!) 
*   Enables **RTL and LTR buttons** [TinyMCE editor]
*   Enables **Create Username containing Persian characters** [Default Wordpress does not allow to use persian characters in usernames]
*   Enables New Admin Typography

== Installation ==

= Installation through WordPress Admin page = 

1. Go to the admin page `Plugins -> Add New`
2. Search for `persian world` and choose the plugin
3. Choose the action `install`
4. Click on `Install now`
5. Activate the plugin after install has finished (with the link or trough the plugin page)
6. :)


= Installation using WordPress Admin page - Upload =

1. Download the plugin zip file
2. Go to the admin page `Plugins -> Add New`
3. Choose the `Upload` link under the `Install Plugins` headline
4. Browse for the zip file and click `Install Now`
5. Activate the plugin after install has finished (with the link or trough the plugin page)
6. ;)


= Installation using ftp =

1. Unzip und upload the files to your `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. That's it.  You're ready to go!
